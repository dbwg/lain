Pluglets

In addition to Lain's built-in functionality, users can write small, sandboxed Lua scripts called pluglets to customize functionality.

Pluglets can be shared on this website and "dropped in" to any server. 

          
--[[ The following is an example of a Lain pluglet ]]--
command "countdown {n=3}" {
  description = "Count down from {n} to zero"

  handler = function(bot, n)
    local m = bot:send(tostring(n))
    for i=n-1,0,-1
      m.edit(tostring(n))
    end
  end
}
